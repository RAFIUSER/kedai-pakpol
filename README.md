# Kedai Pak Pol's Company Profile

Adalah sebuah kedai yang berada di sekitar Politeknik Negeri Sriwijaya tepatnya di Jl. Sungai Sahang, Palembang.

Proyek ini merupakan tugas akhir mata kuliah Pemrograman Dasar Semester 1.

Tugas kami adalah membantu UMKM Lokal untuk meningkatkan visibilitas kedai melalui media digital seperti website landing page yang bisa diakses dengan mudah hanya dengan sebuah browser.

## Tech Statis
- Static Html
- Tailwind CSS
